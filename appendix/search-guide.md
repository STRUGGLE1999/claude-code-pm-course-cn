# 搜索指南

使用下面的搜索框在所有课程页面中查找内容。搜索由 Pagefind 驱动，完全在客户端工作（不需要服务器）。

## 搜索技巧

- **精确短语**：使用引号，如 `project memory`

- **多个词**：搜索 `PRD template` 来查找同时包含这两个词的页面

- **过滤器**：结果显示每个匹配来自哪个部分

## 找不到内容？

如果你找不到你要找的内容：

- 尝试不同的关键词或同义词

- 浏览侧边栏导航

- 查看 [公司背景](https://ccforpms.com/company-context/overview) 了解 TaskFlow 背景

- 从 [入门指南](https://ccforpms.com/getting-started/introduction) 开始了解课程基础
