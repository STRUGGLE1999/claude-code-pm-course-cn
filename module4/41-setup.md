## 什么是 Vibe Coding？

Vibe Coding 是通过描述你想要什么来构建软件，而不是编写代码。

这个词由 Andrej Karpathy 推广——即"完全跟随感觉"的理念。你描述结果，AI 负责实现。

核心技能不是编程，而是：

- 清楚地表达你想要什么
- 给出有效的反馈
- 判断什么时候是对的

你在做指挥，而不是实现。

## PM/工程师的协作模式

这就是非技术背景的创始人与开发者合作的真实方式：

- **你的工作：** 明确你想要什么，给出清晰的反馈
- **他们的工作：** 想办法实现它

在本模块中，你是 PM。Claude 是工程师。

你描述。Claude 构建。你审查。Claude 优化。

这是一项真正通用的专业技能——适用于与设计师、开发者、代理商或任何 AI 工具协作。

## 构建循环

每个项目都遵循相同的循环：

1. **计划** → 明确你要构建什么
2. **构建** → 让 AI 创建它
3. **迭代** → 不断完善直到满意
4. **保存** → 备份到 GitHub
5. **上线** → 部署到互联网

这个循环适用于任何项目——落地页、工具、应用、原型。

## 什么是 Framework？

Framework（框架）是一个预先构建好的基础，让你不用从零开始。

把它想象成房子在添加墙壁和油漆之前的框架结构。它处理常见的模式，让你专注于你的具体应用。

**Next.js** 是我们使用的框架——它在构建网站方面非常流行，并且与 Vercel（我们将要部署的平台）无缝集成。

你不需要理解它，只需要知道它的存在。

## 什么是 JavaScript？

JavaScript 是网站开发中最流行的编程语言。它在浏览器中运行，使网站具有交互性。

你不需要学习它——Claude 会为你编写。
