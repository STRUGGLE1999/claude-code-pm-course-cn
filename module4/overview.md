# 模块4：Vibe Coding

- **完成时间：** 1-2小时

![](https://aka.doubaocdn.com/s/XS0E1wDdzg)

你已经学会了如何操作文件。现在学习如何创建文件。

在本模块中，你将从零开始构建一个真正的 Web 应用并将其部署到互联网上——无需编程经验。你描述想要什么，Claude 负责构建，你不断完善直到满意。

## 你将构建什么

一个性格测试应用，包含：

- 多个问题及自定义逻辑
- 基于答案的性格类型结果
- 任何人都可以访问的真实 URL

同样的流程适用于落地页、工具、仪表盘、原型——任何基于 Web 的项目。

## 你将学到什么

| 课程 | 内容概述 |
|---|---|
| **4.1：设置** | Vibe Coding 的思维方式和项目文件夹设置 |
| **4.2：计划** | 通过访谈流程明确需求 |
| **4.3：构建与迭代** | 搭建脚手架、构建应用，并通过截图进行优化 |
| **4.4：GitHub** | 版本控制和备份 |
| **4.5：上线** | 部署到 Vercel 并获得真实 URL |
